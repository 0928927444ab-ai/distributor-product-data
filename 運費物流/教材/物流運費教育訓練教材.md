## 一、前言：為什麼要了解物流成本

### 1.1 運費對利潤的影響

```
┌─────────────────────────────────────────────────────┐
│                    產品售價 100%                      │
├─────────────────────────────────────────────────────┤
│  產品成本 50%  │  運費 15~30%  │  毛利 20~35%        │
└─────────────────────────────────────────────────────┘
```

**重點觀念**：
- 運費通常佔產品總成本的 **15%~50%**
- 選錯物流，可能讓利潤減少 **10%~20%**
- 了解運費計算 = 掌握成本控制的關鍵

### 1.2 常見問題

| 問題 | 後果 |
|------|------|
| 不知道如何選物流 | 多花冤枉錢 |
| 不清楚計費方式 | 無法預估成本 |
| 忽略額外費用 | 預算超支 |
| 沒有比價習慣 | 錯失省錢機會 |

## 三、計費方式詳解

### 3.1 材積計費（和穎-方式A）

**公式**：
```
運費 = 材積(m³) × 3,000 + 額外費用
```

**材積計算方式**：
```
材積(m³) = 長(cm) × 寬(cm) × 高(cm) ÷ 1,000,000
```

**範例**：
```
外箱尺寸：100cm × 50cm × 40cm
材積 = 100 × 50 × 40 ÷ 1,000,000 = 0.2 m³
運費 = 0.2 × 3,000 = 600 NT
```

---

### 3.2 重量計費（和穎-方式B）

**公式**：
```
運費 = 重量(KG) × 60 + 服務費(貨值×10%) + 其他額外費用
```

> ⚠️ **五金產品強制使用此計費方式，無法選擇材積計費！**

**範例（JY260115 實際案例）**：
```
品項：YG313鋁窗柱+導框（五金產品）
重量：60 KG
貨值：假設 30,000 NT

基礎運費 = 60 × 60 = 3,600 NT
服務費 = 30,000 × 10% = 3,000 NT
調整費用 = 1,657 NT
總運費 = 8,257 NT
```

**服務費計算說明**：
```
服務費 = 產品價值 × 10%

例：貨值 50,000 NT → 服務費 5,000 NT
例：貨值 100,000 NT → 服務費 10,000 NT
```

---

### 3.3 重量計費（飛翔物流）

**公式**：
```
運費 = 重量(KG) × 單價 + 派送費
```

**單價說明**：
| 系列 | 單價 | 適用情況 |
|------|------|----------|
| FAC/FAH | 25 NT/KG | 一般貨品 |
| XB | 55 NT/KG | 快速/特殊 |

**範例**：
```
貨品重量：50 KG（FAC系列）
運費 = 50 × 25 = 1,250 NT
加派送費 = 1,250 + 1,000 = 2,250 NT
```

## 七、注意事項

### 7.1 常見陷阱

| 陷阱 | 說明 | 避免方式 |
|------|------|----------|
| ⭐ **五金想用材積計費** | 五金產品強制重量計費，無法選材積 | **認清規則，直接選飛翔** |
| ⭐ **忘記 10% 服務費** | 和穎所有產品都要加 10% 服務費 | **計算時一定要加上** |
| 不知和穎有兩種計費 | 可能被用較貴方式計費 | **主動詢問計費方式** |
| 忽略額外費用 | 和穎的調整費、稅金可能很高 | 要求完整報價 |
| 只看基本單價 | 飛翔 XB 是 FAC 的 2.2 倍 | 確認適用系列 |
| 沒有比價 | 不同貨品適合不同物流 | 養成比價習慣 |

---

### 7.2 報價確認清單

下單前務必確認：

- [ ] **和穎是用材積還是重量計費？**（五金強制重量）
- [ ] **服務費是否已計入？**（貨值 × 10%）
- [ ] 適用單價（是否有不同費率）
- [ ] 額外費用明細（稅金、調整費）
- [ ] 派送費用
- [ ] 預計到貨時間
- [ ] 是否含稅

---

### 7.3 省錢技巧

1. ⭐ **五金直接選飛翔**：和穎五金強制重量計費 + 10% 服務費，飛翔更划算
2. **主動要求材積計費**：非五金且密度低，向和穎爭取材積計費
3. **飛翔優先考慮**：重量型貨品，飛翔 25 NT/KG 最便宜
4. **併貨出貨**：多種貨品一起出，分攤基本費用
5. **減少包材**：外箱不要太大，減少材積
6. **長期合作**：爭取優惠價格

## 八、練習題

### 練習一：基本計算

**題目**：一批鉸鏈，重量 30 KG，外箱 50×40×30 cm，請計算：
1. 材積是多少？
2. 和穎材積運費？
3. 和穎重量運費？
4. 飛翔運費？
5. 應該選哪家？

<details>
<summary>點擊看答案</summary>

1. 材積 = 50×40×30 ÷ 1,000,000 = **0.06 m³**
2. 和穎材積 = 0.06 × 3,000 = **180 NT**
3. 和穎重量 = 30 × 60 = **1,800 NT**
4. 飛翔 = 30 × 25 = **750 NT**
5. 選 **和穎材積**，最省！

</details>

---

### 練習二：密度判斷

**題目**：以下貨品應該選哪家物流？

| 貨品 | 重量 | 材積 | 密度 | 答案 |
|------|------|------|------|------|
| A. 展示架 | 10 KG | 0.5 m³ | ? | ? |
| B. 門鎖組 | 50 KG | 0.1 m³ | ? | ? |
| C. 鋁窗柱 | 60 KG | 0.3 m³ | ? | ? |

<details>
<summary>點擊看答案</summary>

| 貨品 | 密度計算 | 結果 | 建議 |
|------|----------|------|------|
| A | 10÷0.5=20 | < 50 | **和穎材積** |
| B | 50÷0.1=500 | > 120 | **飛翔** |
| C | 60÷0.3=200 | > 120 | **飛翔**（或比價） |

</details>

### 練習三：實際報價比較

**題目**：收到以下報價，哪個划算？

- 和穎報價：材積 1.2 m³，運費 3,600 + 稅金 500 + 叉車 360 = **4,460 NT**
- 飛翔報價：重量 100 KG，運費 2,500 + 派送 1,000 = **3,500 NT**

<details>
<summary>點擊看答案</summary>

選 **飛翔**，省 960 NT

**分析**：
- 密度 = 100 ÷ 1.2 = 83.3（在 50~120 之間，需比價）
- 和穎額外費用高，實際飛翔較便宜
- **教訓**：要看總價，不能只看單價

</details>

---

### 練習四：和穎計費方式選擇

**題目**：貨品重量 40 KG，材積 0.5 m³，和穎應該選哪種計費？

<details>
<summary>點擊看答案</summary>

- 材積計費：0.5 × 3,000 = **1,500 NT**
- 重量計費：40 × 60 = **2,400 NT**
- 選 **材積計費**，省 900 NT
- 密度 = 40 ÷ 0.5 = 80（< 120，材積划算）

</details>

## 總結

### 核心重點

1. ⭐ **五金走和穎強制重量計費**：無法選材積，建議直接選飛翔
2. ⭐ **和穎加收 10% 服務費**：以產品價值計算，別忘了加上
3. **和穎有兩種計費**：材積 3,000 NT/m³、重量 60 NT/KG
4. **飛翔最便宜**：25 NT/KG（FAC/FAH）
5. **密度決定選擇**：< 50 選和穎材積，> 120 選飛翔
6. **永遠要看總價**，不只看單價
7. **主動確認計費方式**，避免被貴到

> **「極輕和穎材積，很重選飛翔，中間要比價，下單先問計費」**
