## 經銷商制度

### 資格條件（符合任一即適用）

#### 條件一：樂莫集團相關企業 — 摩研、摩築、莫爾睿

| 項目     | 內容                                                      |
| -------- | --------------------------------------------------------- |
| 門檻     | 全品項每月叫貨額達**$100,000**（折扣後含稅價）            |
| 結算週期 | **月結算**，每月 1 日結算上月叫貨額                 |
| 叫貨計價 | 期間叫貨以**5 折**計                                |
| 達標退差 | 結算達標後退**1.5 折**價差（最終 **3.5 折**） |

#### 條件二：購買經銷權

**品牌經銷（全品項）**

| 項目     | 內容                                               |
| -------- | -------------------------------------------------- |
| 門檻     | 全品項年度達**$5,000,000**（折扣後含稅價）         |
| 保證金   | **$1,000,000**                               |
| 結算週期 | **年度結算**，每年 1 月 1 日結算上年度叫貨額 |
| 叫貨計價 | 期間叫貨以**5 折**計                         |
| 達標退差 | 結算達標後退**1.5 折**價差                   |

**產品經銷（單品項）**

| 項目     | 內容                                               |
| -------- | -------------------------------------------------- |
| 門檻     | 單品項年度達**$800,000**（折扣後含稅價）           |
| 保證金   | **$250,000**                                 |
| 結算週期 | **年度結算**，每年 1 月 1 日結算上年度叫貨額 |
| 叫貨計價 | 期間叫貨以**5 折**計                         |
| 達標退差 | 結算達標後退**1.5 折**價差                   |

### 簽約與保證金

- 經銷合約**每年簽約一次**，到期前 30 天通知續約
- 保證金：品牌經銷 **$1,000,000** ／ 產品經銷 **$250,000**（可抵扣最後一批貨款）
- 單次叫貨額如超過保證金，須補足保證金
- 保證金於合約正常結束且無欠款時**全額退還**

### 達標評估

- 品牌經銷 & 產品經銷：**年度結算**（每年 1 月 1 日結算上年度）
- 樂莫集團企業：**月度結算**（每月 1 日結算上月）
- 每季提供一次達標進度通知，方便經銷商自行追蹤

### 未達標處理

- 年度未達標 → 合約到期後**不予續約**，恢復為專案價 5 折
- 不設寬限期，不設降級過渡
- 合約期間內維持叫貨 5 折不變，不中途調整

### 重新申請

- 被取消後滿 **6 個月**可重新申請
- 需重新繳交保證金、重新簽約

### 違約條款

- 合約期間內若有**退票、欠款逾 30 天**，得立即終止合約
- 終止時保證金優先抵扣欠款，餘額退還

## 二、各層級售價計算

| 層級                       | 折數   | 計算方式                             |
| -------------------------- | ------ | ------------------------------------ |
| 牌價                       | 10 折  | 牌價 × 1.00                         |
| 門市滿額 A                 | —     | 滿 $5,000 折 $300                    |
| 門市滿額 B                 | —     | 滿 $10,000 折 $800                   |

| 建材行專案價               | 4.5 折 | 牌價 × 0.45（建材/五金行 + 滿 $500,000，折後含稅價） |

## 三、議價規則

- 享有任何折扣前，須先加入**摩瑪建材 LINE@**
- 一般客人先報**牌價** → 滿 $5,000 折 $300 / 滿 $10,000 折 $800 / 滿 $15,000 享 9 折
- 設計師 / 工程行須出示**乙級技術士證照**，方可享 7.5 折
- 建材 / 五金行須憑**營利事業登記證**（建材或五金相關行業），或為 **BNI 大台中南區綠燈會員**，方可享 6 折
- 專案價須**單筆滿額 $500,000 以上**方可適用 5 折
- 建材 / 五金行具備資格者，**單筆滿額 $500,000 以上**（折後含稅價）可享建材行專案價 **4.5 折**
- 經銷商期間以 **5 折**叫貨，達標後退 **1.5 折**價差（最終 **3.5 折**）
- 不設底價，最低實際折扣為經銷商 3.5 折

## 四、補充規則

- 門市滿額優惠**適用全品項**
- 本模板所有金額皆為**含稅金額**
- 專案價（5 折）、建材行專案價（4.5 折）及經銷商（3.5 折）之售價為**折後含稅價**，不再另加稅
- 經銷商完整資格制度見上方經銷商章節
