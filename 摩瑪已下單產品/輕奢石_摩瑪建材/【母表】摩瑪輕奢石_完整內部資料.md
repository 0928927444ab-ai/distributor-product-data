| 基底類型 | 是否適用 | 備註 |
|----------|:--------:|------|
| 水泥砂漿面 | ✓ 適用 | 需平整、乾燥、無粉化 |
| 水泥板 | ✓ 適用 | 最佳基底 |
| 木夾板（合板） | ✓ 適用 | 需固定穩固、無變形 |
| 石膏板 | ✓ 適用 | 需確認承重 |
| 既有磁磚面 | ✓ 適用 | 需打磨或塗佈界面劑 |
| 金屬面 | ✓ 適用 | 需除鏽、塗佈界面劑 |
| 油漆面 | △ 視情況 | 需打磨去除光滑面 |
| 壁紙面 | ✗ 不適用 | 需先移除壁紙 |

#### 三、黏貼膠材
| 膠材類型 | 適用情境 | 說明 |
|----------|----------|------|
| **輕奢石專用膠**（推薦） | 室內平面/弧面 | 彈性佳、耐候、專為輕奢石設計 |
| 改性矽酮密封膠 | 室內外皆可 | 防水性佳、彈性好 |
| 瓷磚背膠 + 益膠泥 | 既有磁磚基底 | 需先塗背膠增強附著力 |

#### 四、施工步驟
1. **基底檢查** — 確認平整度（落差≤3mm/2m）、乾燥、無粉化脫落
2. **基底處理** — 清除灰塵油污；不平整處以批土整平，待乾燥
3. **放樣定位** — 使用雷射水平儀彈線，確定起始位置與排版方式
4. **裁切備料** — 依需求尺寸裁切輕奢石板，預留1~2mm伸縮縫
5. **塗佈膠材** — 以鋸齒刮刀將專用膠均勻塗於基底（滿塗），厚度約2~3mm
6. **黏貼就位** — 將輕奢石板對齊彈線，由下而上（或由中心向兩側）貼合
7. **排氣壓實** — 使用橡膠滾輪由中心向四周滾壓，排除空氣，確保密實
8. **接縫處理** — 相鄰板材間保留1~2mm縫隙，以填縫劑填補並修飾
9. **收邊美容** — 邊角收邊、清除溢膠、整體清潔
10. **養護期** — 施工完成後24小時內避免碰撞，72小時達到完全固化

#### 五、施工注意事項
- **溫度要求：** 施工環境溫度建議 5°C～35°C
- **濕度要求：** 基底含水率需低於8%
- **弧面施工：** 可利用熱風槍輔助加熱軟化（建議40~60°C），提高彎曲貼合度
- **大板搬運：** 建議兩人以上搬運，避免折彎損壞
- **裁切方向：** 正面朝上裁切，避免崩邊
