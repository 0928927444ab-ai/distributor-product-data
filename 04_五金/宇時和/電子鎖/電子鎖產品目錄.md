## 二、產品通用規格

| 項目 | 規格 |
|------|------|
| 開鎖記錄 | 儲存225條最新開鎖記錄 |
| 電源供給 | 4節/8節 1.5V 5號AA鹼性電池 |
| 適用卡片 | Mifare-1卡S50 / EM卡 / TEMIC5567卡 |
| 欠壓提示 | 電壓低於4.8V時，開鎖有星光提示 |
| 使用壽命 | 30萬次以上 |
| 靜態電流 | ≦18uA |
| 動態電流 | ≦250mA |
| 溫度範圍 | -40℃ ~ +80℃ |
| 相對濕度 | 20%RH ~ 80%RH |

### 產品特點
- 率先研究並成功實現智慧酒店鎖方案
- 門鎖模組均做防水、防潮、防霉處理
- 高低溫、震動、48小時以上老化測試
- 無線聯網方案下，支持APP/微信管理
- 簡便穩定的管理軟件；領先的PDA智能管理系統
- 支持酒店一卡通
